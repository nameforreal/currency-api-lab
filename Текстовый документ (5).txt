Лабораторная работа
Тема: Разработка REST API "Курсы валют"

1. ОБЩЕЕ ОПИСАНИЕ

В данной работе реализовано клиент-серверное приложение для получения информации о курсах валют.

Серверная часть написана на Python с использованием библиотеки Flask.
Клиентская часть использует библиотеку requests для отправки HTTP-запросов.

Приложение работает по архитектуре REST API и обменивается данными в формате JSON.

--------------------------------------------------

2. СЕРВЕРНАЯ ЧАСТЬ (currency_api_server.py)

Сервер отвечает за обработку запросов и возврат данных.

Используется структура данных CURRENCY_DATA, которая хранит:
- код валюты (USD, EUR)
- название валюты
- текущий курс
- историю курсов

Пример структуры:

USD:
- current_rate: текущий курс
- history: список дат и значений курса

--------------------------------------------------

3. ОПИСАНИЕ ENDPOINTS

1) /api/currencies
Метод: GET

Назначение:
Возвращает список всех валют и их текущие курсы.

Как работает:
Сервер просто возвращает весь словарь CURRENCY_DATA в формате JSON.

--------------------------------------------------

2) /api/currency/current?code=USD
Метод: GET

Назначение:
Возвращает текущий курс выбранной валюты.

Как работает:
- Получает параметр code из URL
- Проверяет, есть ли такая валюта
- Если есть — возвращает курс
- Если нет — возвращает ошибку 404

--------------------------------------------------

3) /api/currency/history?start=...&end=...
Метод: GET

Назначение:
Возвращает историю курсов за указанный период.

Как работает:
- Получает даты start и end
- Проходит по всем валютам
- Фильтрует записи по диапазону дат
- Возвращает список результатов

--------------------------------------------------

4. КЛИЕНТСКАЯ ЧАСТЬ (currency_api_client.py)

Клиент отправляет запросы к серверу и выводит результат.

Реализовано меню:

1 — получить список валют
2 — получить курс по коду
3 — получить историю
4 — выход

--------------------------------------------------

5. ПРИНЦИП РАБОТЫ

1. Запускается сервер
2. Сервер начинает слушать порт 5000
3. Клиент отправляет HTTP-запросы
4. Сервер обрабатывает запрос
5. Возвращает JSON
6. Клиент выводит результат в консоль

--------------------------------------------------

6. ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ

- Python
- Flask (сервер)
- requests (клиент)
- HTTP
- JSON

--------------------------------------------------

7. ВЫВОД

В ходе работы был разработан REST API для работы с курсами валют.
Реализованы три основных endpoint:
- получение всех валют
- получение курса по коду
- получение истории курсов

Приложение демонстрирует взаимодействие клиента и сервера и работу с HTTP-запросами.
